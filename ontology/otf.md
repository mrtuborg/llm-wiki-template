# OTF (Open Terminology Framework)
# External Ontology Categories

OTF is an external semantic classification system used to annotate internal
wiki node types with standardized conceptual categories.

OTF categories do NOT replace internal node types. They provide an additional
layer of semantic normalization for interoperability and reasoning.

## OTF Categories

- OTF:Object
- OTF:Process
- OTF:Structure
- OTF:Procedure
- OTF:Condition
- OTF:Invariant
- OTF:Abstraction
- OTF:Summary
- OTF:Integration
- OTF:Choice
