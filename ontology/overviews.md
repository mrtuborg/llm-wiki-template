# Overview — Ontology Definition

**OTF Category:** `OTF:Summary`  
**Tagline:** Human-readable explanation of one Concept

## Definition

Page explaining a single Concept: summary, context, examples, limitations. Written for a human reader.

- `Concept → Overview (summarized_into)`

## Allowed Outbound Edges

- Overview of Idempotency in OTA retry logic.
